# Brazillian Jiu-Jitsu Stats Scraper

### Requirements
* Python 2.6+
* Pip
* BeautifulSoup library

### Installing
```
git clone https://github.com/mikecoram/bjj-stats-scraper.git
```

### Installing Dependencies
```
sudo pip install beautifulsoup4
```

### Running
```
python scraper.py
```
